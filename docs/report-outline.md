# DLAW207 Project Report Outline
## OpenRights SL - Intellectual Property Tool for Sierra Leone

**Course:** IT Law and IPR (DLAW207)
**Institution:** Limkokwing University, Sierra Leone
**Semester:** 06 (March-July 2026)

### 1. Introduction
- Background: The gap in legal protection for Sierra Leonean creators.
- Objective: Provide an accessible, open-source compliance tool to help musicians, coders, and writers protect their work using IP frameworks.

### 2. Legal Frameworks
- Intellectual Property Rights (Copyright, Trademarks, Patents).
- Open Source Licenses (MIT, Apache 2.0, GPL v3, CC-BY, CC-BY-NC).
- Integration with Sierra Leone Cyber Security Act 2021.
- GDPR Data Minimisation Principles (Article 5.1.c).

### 3. Application Features
- License Recommender Questionnaire.
- Document Generator (LICENSE.txt, Privacy Policy, PDF Certificate).
- Contract Analyzer & License Scanner (Dependency Analysis).
- Creator Registry as a Digital Public Good.
- AI Chatbot Assistant for Legal Guidance.

### 4. Technical Implementation
- Client-side application using HTML5, CSS3, Vanilla JavaScript.
- No Backend, Privacy by Design (localStorage handling).
- Integrations: Groq AI API, JSONBin, jsPDF, QRCode.js.
- Progressive Web App (PWA) with offline capabilities.

### 5. SDG Alignment
- SDG 16: Peace, Justice & Strong Institutions (Legal Accessibility).

### 6. Conclusion
- Impact of the platform on local creators.

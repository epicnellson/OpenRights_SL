License Glossary

Learn key open source licensing terms in plain English.

Copyleft

A licensing requirement that any derivative work (modified version) must be distributed under the same license as the original. Think of it as "share-alike."

Sierra Leone example: GPL ensures Freetown app developers must share their code if they distribute modified versions.

Permissive License

A license with minimal restrictions. Users can use, modify, and even sell your code as long as they include attribution.

Sierra Leone example: MIT license lets Bo developers build commercial products without sharing source code.

Patent Grant

An explicit license from contributors giving you rights to any patents they include in their code.

Sierra Leone example: Apache 2.0 protects you if a contributor's code uses a patented method.

Attribution

Requirement to credit the original creator when using or distributing their work.

Sierra Leone example: Always mention "Made by [Name] under MIT License" in your project README.

Derivative Work

A new work based on or adapted from an existing copyrighted work.

Sierra Leone example: Adding new features to someone's MIT-licensed library creates a derivative work.

Open Source

Software with source code that is publicly available for anyone to view, modify, and distribute.

Sierra Leone example: Linux, Python, and Node.js are open source projects.

Digital Public Good (DPG)

Open source technologies, data, and standards that are freely available and can improve people's lives globally.

Sierra Leone example: MIT-licensed libraries used in government digital services qualify as DPGs.

Intellectual Property (IP)

Creations of the mind — inventions, literary works, designs, symbols, names — that can be legally protected.

Sierra Leone example: Your app code, website designs, and written tutorials are all intellectual property.

Copyright vs Trademark vs Patent

Copyright protects original works (code, writing). Trademark protects brand names/logos. Patent protects inventions.

Sierra Leone example: Your code has copyright automatically. "OpenRights SL" would need trademark registration.
